### Overview

Shopper is an e-commerce platform for purchasing fashion apparel and accessories for all ages. It is a one-stop online apparel shop that imitates Shopify websites with built-in features, including add-to-cart items and payment integration. Shopper has a separate application for users and admins to isolate. There are admin-only features for product management, such as adding new products and removing existing ones. Customers can select a variety of apparel for men, women, and kids using the cart feature and later purchase the product via the Stripe payment gateway.

### Objectives

- Develop an e-commerce website with add to cart and Stripe payment gateway integration.
- Create a separate admin webpage for product management.
- Design a responsive website for better accessibility across devices.

### Tech Stack

- **Frontend:** ReactJS, CSS, JavaScript
- **Backend:** Express, NodeJS
- **Tools:** VS Code, Postman
- **Services:** Stripe

### Features

- User authentication and admin control.
- Shopping cart and Stripe payment gateway.
- Modern and responsive UI.
